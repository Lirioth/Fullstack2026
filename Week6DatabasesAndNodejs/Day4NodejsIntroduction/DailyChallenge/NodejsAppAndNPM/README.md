# Daily Challenge — Node.js Modules, NPM & Files

Mini‑project demonstrating:
- A basic CommonJS module
- Using an NPM package (chalk)
- File I/O with `fs`
- A final script that integrates everything

## Setup
```bash
cd daily-challenge
npm install
```

## Run
```bash
# Task 1
node task-1/app.js

# Task 2
node task-2/app.js

# Task 3
node task-3/app.js

# Challenge (integration)
node challenge.js
```
